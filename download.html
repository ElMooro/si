Install notes
• Mac: open the .dmg and drag JustHodl AI to Applications. First launch: right-click → Open (the app isn't yet notarized, so macOS asks once).
• Windows: run the .exe installer. SmartScreen may warn on first run (unsigned) — click "More info → Run anyway".
• Linux: mark the AppImage executable (chmod +x) and run, or install the .deb.
• An internet connection is required — the app loads live market data and your tools.
