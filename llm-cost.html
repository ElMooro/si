Controls
Governance is tuned live via SSM — no redeploy needed (5-min propagation):
• Mode /justhodl/llm/mode = normal · economy (force reason/critical → cheapest model) · off (all engines fall back to deterministic output, zero LLM spend).
• Daily budget /justhodl/llm/daily-budget-usd — hard ceiling; over it, engines return deterministic output for the rest of the day.
• Prices /justhodl/llm/prices — optional JSON override of per-model $/1M-token.
The three savings levers, in order of impact: content cache (byte-identical requests skip the model entirely — the bulk of the cut), economy routing (cheapest capable model), and the budget cap (runaway guard). All governance is fail-safe: any error degrades to a normal model call, so it can never break an engine.
