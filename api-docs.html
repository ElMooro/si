Quick start

# curl
curl -s https://justhodl-data-proxy.raafouis.workers.dev/data/sector-groups.json | head -c 400

# python
import requests
g = requests.get("https://justhodl-data-proxy.raafouis.workers.dev/data/themes.json").json()
for t in g["themes"]: print(t["theme"], t["perf"]["m1"], t["leader_1m"])

# javascript
const d = await (await fetch("https://justhodl-data-proxy.raafouis.workers.dev/data/market-map.json")).json();
console.log(d.breadth, d.tiles.length);

CSV exports

The Map, Groups and Themes pages each carry one-click ⬇ CSV buttons (tiles, sectors, industries, theme members) — Sheets-ready.

Authenticated compute API

Interactive endpoints (AI chat, screeners, portfolio admin) live at api.justhodl.ai behind keyed tiers: FREE 100 req/hr · PRO 5,000 req/hr · ENTERPRISE unlimited. Keys are issued per account; rate state is enforced server-side. Public JSON briefs above never require a key.

Briefs are research outputs, refreshed on engine schedules, versioned in each payload (`version`, `generated_at`). Schemas can evolve — pin to fields you use. Research, not advice.
