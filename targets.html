🎯 Sustained Targets · 30-Day Front-Run Leaderboard

Which names keep getting front-run? Each daily digest (09:00 + 21:00 UTC) captures the top-targeted assets from the equity sniffer's 7-day rolling stats and the top-targeted instruments from the macro sniffer's 7-day stats. This page aggregates those observations into a 30-day rolling leaderboard so you can distinguish sustained patterns (a name appearing in 8+ digests with high max-events-per-window — true smart-money flag) from one-off flags (a name appearing once or twice and disappearing). Up to 2 observations contribute per name per day (one per digest session). Names not seen in 30+ days auto-prune.

30-day rolling window 🎯 equity assets (39-feed sniffer) 🏛 macro instruments (20-pillar sniffer) ● = active in last 7d ○ = cooling off (8-30d) updated each digest · 09:00 + 21:00 UTC
What it means when a name is high on the leaderboard:

n_digest_appearances — the bar + count. Each digest that flagged this name in its top-targeted list contributes one count. With 2 digests/day × 30 days = 60 maximum possible. A name at 8×+ over 30 days = present in 13%+ of digests = sustained pattern. A name at 1-2× = recent one-off.

max events / window — the peak n_times observed in any single 7-day rolling window. The equity sniffer might flag PEP because it appeared 3 times in 7 days as the top setup target. That 3 is the max for that window. Across multiple digests, the highest 3 sticks as the max even if subsequent windows show 1 or 2.

last seen — relative to today. If today or 1d ago, the green ● recent indicator lights up. 8d or older means the pattern is cooling — name still tracked but flagged with grey ○. After 30 days of no observations, the entry prunes automatically.
